Thu tu doc code hieu qua nhat

Nguyen tac: Doc tu tong the den chi tiet, tu Server truoc roi den Agent. Moi file, hieu no lam gi, nhan data gi, tra data gi.

Phase 1: Hieu Server (doc truoc)

  1. server/app.py De
    537 dong
    Entry point. Hieu cach Flask khoi dong, dang ky blueprint, tao ket noi MongoDB. Doc xong file nay ban biet toan bo he thong co nhung gi.
  2. server/models/user_model.py De
    ~150 dong
    Hieu cau truc du lieu User (username, password_hash, role). Day la nen tang cua RBAC. Cac model khac (agent, log, whitelist) tuong tu.
  3. server/models/agent_model.py De
    Hieu Agent duoc luu the nao trong database: agent_id, hostname, device_id, status, group_id, last_heartbeat.
  4. server/services/jwt_service.py Trung binh
    ~300 dong
    Quan trong! Hieu cach tao token, verify token, refresh token, revoke token. Day la "chia khoa" cua toan bo he thong auth.
  5. server/middleware/auth.py Trung binh
    442 dong
    Hieu decorator @require_api_key, @require_jwt. Moi request Agent gui den deu di qua day. Ban se hieu "request vao nhu the nao".
  6. server/middleware/rbac.py Trung binh
    284 dong
    Hieu @require_login (doc JWT tu cookie), @require_admin, @require_permission. Moi request tu web dashboard di qua day.
  7. server/controllers/agent_controller.py De
    Doc 1 controller la hieu tat ca. Hieu luong: request vao → middleware kiem tra → controller xu ly → goi service → tra response.
  8. server/controllers/admin_auth_controller.py De
    Hieu luong dang nhap Admin/Teacher: POST login → check password → tao JWT → set cookie → redirect dashboard.
  9. server/config/rbac_config.py De
    File nay dinh nghia "admin duoc lam gi, teacher duoc lam gi". Doc xong hieu toan bo he thong phan quyen.

Phase 2: Hieu Agent (doc sau)

  1. agent/core/agent.py De
    133 dong
    Singleton class chinh. Hieu agent_id duoc tao tu hardware hash nhu the nao, state machine cua agent.
  2. agent/core/lifecycle.py Trung binh
    FILE QUAN TRONG NHAT cua Agent. Hieu thu tu khoi dong: register → token → whitelist sync → firewall → sniffer → heartbeat.
  3. agent/core/registry.py De
    Agent gui POST /api/agents/register voi API Key, nhan lai agent_id + JWT tokens.
  4. agent/core/token_manager.py Trung binh
    Background thread tu dong refresh JWT truoc khi het han. Neu refresh that bai 3 lan → dang ky lai.
  5. agent/capture/sniffer.py Kho
    197 dong
    Dung Scapy bat goi tin TCP/UDP. Trich xuat domain tu HTTP Host, HTTPS SNI, DNS query. Can hieu networking.
  6. agent/firewall/manager.py Kho
    667 dong
    File dai nhat, phuc tap nhat. Quan ly Windows Firewall: Default Deny, ALLOW rules, domain-to-IP resolution.
  7. agent/whitelist/sync.py De
    GET /api/whitelist/agent-sync, nhan danh sach domain/IP, cap nhat local state.
  8. agent/services/heartbeat.py De
    POST /api/agents/heartbeat moi 20 giay. Don gian, de hieu.
  9. agent/logging_module/sender.py De
    Gom log thanh batch, POST /api/logs. Co retry logic khi that bai.

Luong Request chi tiet

1

Agent Startup (Khoi dong)

▶
Agent khoi dong │ ▼ POST /api/agents/register Header: X-API-Key: abc123 Body: { hostname: "PC-LAB-01", device_id: "sha256-of-hardware", ip_address: "192.168.1.100", platform: "Windows", os_info: "Windows 10", agent_version: "1.0.0", admin_privileges: true } │ ▼ Server tra ve: { success: true, data: { agent_id: "agent-abc123", jwt: { access_token: "eyJ...", // 24h refresh_token: "eyJ...", // 7 ngay } } } │ ▼ Agent luu agent_id + JWT tokens → Tu gio tro di, moi request dung JWT thay vi API Key
2

Whitelist Sync (Dong bo danh sach trang)

▶
Agent can biet domain nao duoc phep │ ▼ GET /api/whitelist/agent-sync?agent_id=abc123 Header: Authorization: Bearer eyJ... │ ▼ Server tra ve: { success: true, domains: [ { value: "google.com", type: "domain", scope: "global" }, { value: "school.edu.vn", type: "domain", scope: "group" }, { value: "*.microsoft.com", type: "pattern", scope: "global" } ], count: 42, timestamp: "2026-03-12T..." } │ ▼ Agent lam gi voi data nay? 1. Luu vao local state 2. Phan giai domain → IP (DNS lookup) google.com → 142.250.80.46 school.edu.vn → 103.28.38.1 3. Tao ALLOW rules trong Windows Firewall netsh advfirewall firewall add rule name="FC_142.250.80.46" ... 4. Tat ca IP khac bi BLOCK boi Default Deny policy
3

Packet Capture → Log (Bat goi tin → Bao cao)

▶
User tren may mo trinh duyet, truy cap facebook.com │ ▼ Windows Firewall: facebook.com IP khong co ALLOW rule → BLOCK │ ▼ Scapy Sniffer bat duoc goi tin DNS query "facebook.com" Filter: "tcp port 80/443 or udp port 53" │ ▼ Extractor trich xuat: { domain: "facebook.com", src_ip: "192.168.1.100", dest_ip: "157.240.1.35", protocol: "HTTPS", port: 443, action: "BLOCKED" } │ ▼ LogSender gom vao batch (doi du 100 log hoac 2 giay) │ ▼ POST /api/logs Header: Authorization: Bearer eyJ... Body: { logs: [ { timestamp, agent_id, action: "BLOCKED", domain: "facebook.com", ... }, { timestamp, agent_id, action: "ALLOWED", domain: "google.com", ... }, ... (batch nhieu log cung luc) ] } │ ▼ Server luu vao MongoDB → Emit SocketIO → Dashboard cap nhat real-time
4

Heartbeat (Tin hieu song)

▶
Moi 20 giay, Agent gui: │ ▼ POST /api/agents/heartbeat Header: Authorization: Bearer eyJ... Body: { agent_id: "abc123", timestamp: "2026-03-12T10:30:00+07:00", metrics: { memory_percent: 45.2, disk_percent: 62.1, uptime_seconds: 86400 }, status: "active" } │ ▼ Server xu ly: 1. Update agent.last_heartbeat = now 2. Update agent.metrics 3. Tinh status: - active: heartbeat < 60 giay truoc - inactive: 60-300 giay - offline: > 300 giay (5 phut) 4. Emit SocketIO "agent_heartbeat" → Dashboard cap nhat trang thai real-time
5

Token Refresh (Lam moi token)

▶
Background thread kiem tra moi 60 giay: "Access token con han khong?" │ ▼ Con duoi 5 phut → Refresh! │ POST /api/auth/refresh Body: { refresh_token: "eyJ...", rotate: false // true = doi ca refresh token } │ ▼ Thanh cong: { access_token: "eyJ-MOI...", expires_in: 86400 // 24h nua } → Luu token moi, tiep tuc binh thuong That bai (HTTP 401): → Thu lai, neu that bai 3 lan → GOI LAI register tu dau (toan bo flow #1)
6

Admin/Teacher Login (Dang nhap web)

▶
Teacher mo trinh duyet, vao trang /login │ ▼ POST /api/admin/auth/login Body: { username: "teacher1", password: "pass123" } │ ▼ Server kiem tra: 1. Tim user trong MongoDB 2. bcrypt.compare(password, password_hash) 3. Check user.is_active == true 4. Check khong bi lock (< 5 lan login sai) │ ▼ Tao JWT voi token_for="admin_user" Set-Cookie: access_token=eyJ... (httpOnly, 24h) Set-Cookie: refresh_token=eyJ... (httpOnly, 7 ngay) │ ▼ Redirect → / (Dashboard) │ ▼ Dashboard JavaScript goi API: GET /api/agents ← Cookie tu dong gui theo GET /api/logs/stats ← Middleware doc cookie, kiem tra role GET /api/whitelist ← Neu Teacher: chi tra ve data cua group minh │ ▼ SocketIO ket noi → Nhan cap nhat real-time

Tong hop: 1 request di qua nhung gi?

▶
Vi du: GET /api/agents (Teacher xem danh sach agent) Browser → Flask Router → Middleware → Controller → Service → Model → MongoDB 1. Browser gui request GET /api/agents Cookie: access_token=eyJ... 2. Flask Router Tim route match: /api/agents → agent_controller.list_agents 3. Middleware chay truoc controller: a. @inject_current_user (rbac.py) → Doc cookie "access_token" → Verify JWT (kiem tra chu ky, het han, revoked) → Kiem tra token_for == "admin_user" → Tim user trong DB → Set g.current_user = {username, role, ...} → Set g.current_role = "teacher" 4. Controller (agent_controller.py) → Kiem tra g.current_role == "teacher" → Neu teacher: lay danh sach group_ids cua teacher → Goi agent_service.list_agents(filter={group_id: {$in: group_ids}}) 5. Service (agent_service.py) → Goi agent_model.find(filter, limit, skip) 6. Model (agent_model.py) → db.agents.find({group_id: {$in: [...]}}).limit(20).skip(0) 7. MongoDB tra ve documents 8. Response nguoc lai: MongoDB → Model → Service → Controller → JSON Response { agents: [{agent_id, hostname, status, ...}, ...], total: 15, pagination: { page: 1, limit: 20 } }

Tat ca API Endpoints

Agent → Server (Agent gui request)

POST /api/agents/register API Key
Agent dang ky lan dau. Gui thong tin may, nhan lai agent_id + JWT.
POST /api/agents/heartbeat JWT
Agent gui tin hieu song moi 20 giay kem metrics (RAM, disk, uptime).
GET /api/whitelist/agent-sync JWT
Agent dong bo danh sach domain/IP duoc phep. Params: agent_id, since.
POST /api/logs JWT
Agent gui batch log (ALLOWED/BLOCKED). Body: { logs: [...] }
POST /api/auth/refresh None
Refresh JWT token. Body: { refresh_token, rotate: bool }
POST /api/auth/logout JWT
Revoke tokens cua agent.
POST /api/auth/verify None
Kiem tra token co hop le khong. Body: { token, token_type }

Web Dashboard → Server (Admin/Teacher request)

POST /api/admin/auth/login None
Dang nhap. Body: { username, password }. Tra ve: set httpOnly cookies.
POST /api/admin/auth/logout Cookie
Dang xuat. Xoa cookies, revoke tokens.
GET /api/admin/auth/me Cookie
Lay thong tin user dang dang nhap.
GET /api/agents Cookie
Danh sach agents. Teacher chi thay agents trong group minh. Params: limit, skip, status, group_id.
GET /api/agents/statistics Cookie
Thong ke: total, active, inactive, offline. Teacher: chi group minh.
DELETE /api/agents/:id Cookie
Xoa agent. Teacher khong co quyen nay.
PATCH /api/agents/:id/group Cookie
Chuyen agent sang group khac. Teacher phai so huu ca 2 group.
GET /api/whitelist Cookie
Danh sach whitelist. Teacher: global + group minh. Params: search, group_id, limit.
POST /api/whitelist Cookie
Them domain/IP. Teacher chi them vao group minh, khong the them global.
DELETE /api/whitelist/:id Cookie
Xoa entry. Teacher chi xoa trong group minh.
GET /api/logs Cookie
Xem logs. Teacher chi thay logs cua agents minh. Params: action, level, agent_id, time_range.
GET /api/groups Cookie
Danh sach groups. Teacher chi thay group minh tao.
GET /api/users Cookie
Danh sach users. Chi Admin moi co quyen.

Lo trinh hoc 45-60 ngay

Meo: Moi giai doan, doc file tuong ung trong project truoc, roi tu viet lai phien ban don gian. Khong copy-paste!

Ngay 1-10: Python + Git co ban

Hoc: bien, ham, class, decorator, threading, file I/O, pip.

Muc tieu: Doc hieu duoc agent/config/defaults.py va server/models/user_model.py

Viet lai: Tao class Agent don gian voi __init__, start(), stop()

Ngay 11-20: Flask + MongoDB

Hoc: HTTP methods, Flask route/blueprint, PyMongo CRUD, MVC pattern.

Doc: server/app.py → server/controllers/agent_controller.py → server/services/agent_service.py

Viet lai: Flask app voi 4 route CRUD cho Agent (GET, POST, PUT, DELETE)

Ngay 21-27: JWT + RBAC

Hoc: BCrypt, JWT (access/refresh), decorator pattern, middleware.

Doc: server/services/jwt_service.py → server/middleware/auth.py → server/middleware/rbac.py

Viet lai: He thong login voi JWT + 2 role (admin/teacher)

Ngay 28-34: Frontend + SocketIO

Hoc: HTML/CSS, JavaScript fetch API, Jinja2 template, Socket.IO client.

Doc: server/templates/ (dashboard.html, login.html, agents.html)

Viet lai: Login page + Dashboard hien thi danh sach agent

Ngay 35-44: Networking + Scapy

Hoc: TCP/IP, HTTP/HTTPS, DNS, Scapy sniff/filter, Wireshark.

Doc: agent/capture/sniffer.py → agent/capture/extractors.py

Viet lai: Script bat 100 goi tin, trich xuat domain tu DNS + HTTPS SNI

Ngay 45-54: Agent Core + Firewall

Hoc: Windows netsh, firewall policies, CustomTkinter GUI.

Doc: agent/core/lifecycle.py → agent/firewall/manager.py

Viet lai: Agent dang ky voi server, sync whitelist, tao firewall rules

Ngay 55-60: Tich hop + Test

Ghep Server + Agent. Test end-to-end: register → sync → capture → block → log.

Muc tieu cuoi: Demo toan bo he thong chay tron ven
